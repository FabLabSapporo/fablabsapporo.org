# fablabsapporo.org
Website &amp; Content for fablabsapporo.org

# VitePress Default

Dev
`npm run docs:dev`

Build
`npm run docs:build`

Preview Build
`npm run docs:preview`


## Config

`config.js` lives in `.vitepress/config.js`

```js
export default {
  lang: 'en-US',
  title: 'VitePress Test',
  description: 'Vite & Vue powered static site generator.',
  base: '/',
  cleanUrls: true,
  outDir: './dist' // distribtion folder
}

```
